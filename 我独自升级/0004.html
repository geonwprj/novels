我独自升级

我独自升级 第4节

「第一次見到好似噉嘅地底城。」

「覺得有啲… … … 唔多對路。」

啲獵人好小心噉觀察住周圍嘅環境。

門入面嘅空間好似一座古老嘅廟宇噉。

呢度好似俾人埋喺地底好耐噉，係一座陰暗又潮濕嘅神殿；啲青苔同埋雜草好似癬噉分佈喺地板、牆身同埋天花板上面。

有幾個獵人見到噉嘅情況就即刻退返後，身體有啲微微震騰騰。

「你哋唔覺得呢度有啲怪怪哋咩？」

「總之覺得好似有嘢望住自己噉！」

三、四個夠薑嘅獵人唔理隊伍後面啲驚青嘅人，繼續向前行入去。

「嘖！唔好講埋啲咁嘅衰嘢，知唔知呀？」

「我哋速戰速決，然後各回各家啦。」

門入面嘅空間異乎尋常噉闊落。

巨大嘅圓形屋頂籠罩住喺人哋嘅頭頂上面。

呢度嘅空間大概有幾個首都奧林匹斯體育館加埋咁大，可能，仲要大過佢哋。

然而反而莫名其妙噉俾人一種抖唔到氣嘅感覺。

原因好明顯。

「嗰… … … 嗰個係… … … 」

「嗰啲咁邪嘅嘢唔會就係大佬掛？」

喺呢個地方嘅最深處，一個超出想像嘅龐然大物坐喺同佢一樣巨大嘅寶座上面。

龐然大物唔係乜嘢嘢，而係一座巨石神像！

「大佬呀！」

「哇噻！」

啲獵人忍唔住噉驚嘆。

最先浮現喺程肖宇腦海嘅係紐約嘅自由神像。如果自由神像坐喺凳上面，會唔會都係咁大呢？

不過，自由神像係個女像，而呢度坐住嘅係個男像啫嘛。

‘呢個神像可能會再大啲… … … ’

喺神像前面，啲獵人緊張噉吞咗啖口水。驚死呢個巨大嘅神像就係呢個地底城入面嘅終極大佬。

‘……’

好彩嘅係，神像一動都唔動。

「呼… … … 噓… … … 」

就連宋志國都長長噉吁咗啖氣。

「好啦，大家，散水啦。」

啲放下心頭大石嘅獵人各自散開，開始喺附近搵嘢。

「呢度好似冇魔獸喎？」

「我都覺。」

「唔好講魔獸啦，我連一隻蟲都冇見到。」

雖然神像所在嘅空間好大，但係入面嘅結構就好簡單。

喺牆上面，掛住數唔清嘅火把；喺牆邊，有好多稍微高過人少少嘅石像靜靜噉企喺度。

「幾靚喎！」

「好似啲藝術品噉！」

企喺牆邊嘅石像形態各異。

每一尊石像都攞住唔同嘅嘢。有啲攞住武器，有啲攞住書，有啲攞住樂器，甚至有啲攞住火把。

「呢個就好似… … … 」

「好似神聖嘅寺廟入面啲雕像噉。」

宋志國搶先講咗王建設想講嘅嘢。

宋志國望住地下，好似發現咗啲乜嘢噉。

「嗯？呢個… … … 難道係魔法陣嚟嘅？」

神殿嘅中央畫住一個前所未見嘅魔法陣。

就喺嗰陣時。

「志國叔！呢度好似寫咗啲嘢。你過嚟睇吓啦！」

有個獵人喺角落頭發現咗一尊唔同嘅石像，就大聲嗌。

啱啱觀察緊魔法陣嘅宋志國企起身，向嗰尊石像行過去。

其他獵人都圍喺嗰嚿石像周圍。

呢尊石像唔同嘅地方就係，佢有一對翼，而且手入面有一塊石板。

呢塊石板上面刻住嘅文字吸引咗啲獵人嘅注意。

宋志國望咗一眼石板，自言自語噉話。

「呢啲係如尼文。」

如尼文。

呢種文字喺地球上面係冇嘅，係地底城嘅世界獨有嘅。而且淨係得啲覺醒咗魔法能力嘅獵人先至睇得明呢種文字。

「卡特農神廟嘅法則。」

宋志國讀咗第一句出嚟。

程肖宇心悒悒噉聽住宋志國讀出嚟嘅石板上面嘅內容。

突然間，有人拉咗吓佢嘅手臂。

擰轉頭一睇，曉希塊面青嗮噉企喺嗰度。

第三章 進攻嘅開始

睇到曉希嘅面色，程肖宇嚇咗一跳。

「做乜嘢呀？係邊度唔舒服呀？」

「嗰… … … 嗰個… … … 」

程肖宇望向曉希嘅手指尖所指住嘅地方。

係嗰尊巨大嘅神像，更準確嚟講，係神像嘅面口。

程肖宇覺得奇怪噉擰歪個頭，因為神像睇起嚟同啱啱一樣。

「……？」

曉希口窒窒噉講：「佢，佢嘅眼珠郁咗郁，佢啱啱喺度望住我哋。」

「乜嘢？」

程肖宇抬起頭望向神像，但係神像睇起嚟同啱啱一樣，冇任何變化。

「應該係你睇錯咗掛… … … 」

然而，曉希好似冇聽到佢講嘅嘢噉，佢低住頭，捉實程肖宇嘅手臂，身體震騰騰得仲犀利。

‘等等！’

就喺呢個時候，程肖宇都察覺到唔對路。周圍靜到出奇。

「啲聲冇嗮… … … 」

火把上面啲火燄燒嘅聲都慢慢消失咗。

「第一，」

然而，宋志國嘅聲音竟然都仲係好清楚噉傳入耳仔。

「敬仰神。第二，讚美神。第三，證明你嘅信仰。違反呢條法則嘅人，將會永遠冇可能生勾勾噉行得出呢個地方。」

就喺呢個時候。

嘭！

突然之間傳嚟嘅爆炸聲令到喺現場嘅每個人好似發夢發醒噉。

「乜嘢… … … 乜嘢情況… … … 」

「呢個係乜嘢聲呀？」

程肖宇最先注意到周圍發生嘅唔尋常嘅變化。

佢嘅感官一早就高速噉運轉緊，一瞬間就分辨到巨響從邊度嚟。

「門！度門閂埋咗啦！」

程肖宇一嗌出聲，所有人都望向入口。打開咗嘅石門已經俾人牢牢噉閂埋咗。

「弊家伙！我頂唔順啦。」

第一個反對繼續探索地底城嘅獵人大大聲鬧住噉大步走向石門。

「我要返屋企！乜嘢終極大佬，乜嘢寶藏，你哋自己搵啦！」

嗰個獵人好似表示抗議噉怒睥住宋志國，然後擰轉身嬲爆爆噉捉住咗門環。

就喺嗰一瞬間，宋志國瞪大咗雙眼，

「唔好呀！」

唰！
